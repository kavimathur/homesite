# Teapot Labs - Coming Soon Page

A simple and elegant "coming soon" landing page for Teapot Labs featuring an animated teapot pouring tea into cups.

## Features

- Animated teapot pouring effect
- Moving cups animation
- Responsive design
- Custom color scheme

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Setup

1. Clone the repository
2. Open `index.html` in your web browser

## Color Scheme

- Background: #cccc97
- Text and Icons: #4a4a35
